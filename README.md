# 🛒 D-Mart Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
This project presents a Power BI dashboard analyzing D-Mart sales data to uncover trends in revenue, product performance, and customer behavior.

## 🎯 Objectives
- Track overall sales performance
- Identify top-selling products
- Analyze category-wise contribution
- Monitor regional sales trends

## 🛠 Tools Used
- Power BI
- Data Cleaning
- DAX (Data Analysis Expressions)
- Data Visualization

## 📊 Key Insights
- Top-performing product categories
- Sales trends over time
- Region-wise revenue distribution
- Customer purchasing patterns

## 📁 Files Included
- `.pbix` Power BI dashboard file
- Dataset (if available)
- Dashboard screenshots
